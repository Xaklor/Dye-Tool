Dye tool made by Xaklor

DyeTool will automatically update when first run using static.drips.pw
-re-launch the application afterwards to use

use WASD to move the selected character, hold space to play attacking animations. 

left click a dye or cloth pattern to select it for the clothing/large option, right click for the accessory/small option. 
left/right clicking the alpha pattern resets the respective dye/cloth.

DyeTool does not update automatically update itself whenever DECA adds changes,
click the Update! button to update DyeTool with the current version of the game.
